## Reddit / Dev Post

I made a small Electron desktop pet for Windows using plain HTML, CSS, and JavaScript.

It is a transparent always-on-top overlay rather than a normal website. The pet can be dragged around, fed, put to sleep, played with, and it saves stats locally in JSON. I also added a hidden task panel with quick notes, reminders, website opening, folder opening, and a 25-minute focus timer.

The fun part: it has Naruto-inspired chibi modes for Sakura Haruno, Tsunade, and Hinata Hyuga, all built with CSS so the app still works without image assets.

Repo: https://github.com/soloescapeyt-create/DesktopPet
