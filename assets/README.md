# RomDesktopPet assets

The app looks for these optional local images:

- `pet-idle.png`
- `pet-happy.png`
- `pet-sad.png`
- `pet-eating.png`
- `pet-sleeping.png`
- `pet-working.png`

When an image is missing, RomDesktopPet automatically uses the built-in CSS pet.

The built-in CSS pet supports Sakura, Tsunade, and Hinata chibi-inspired themes from the task panel settings, so image files are optional.
